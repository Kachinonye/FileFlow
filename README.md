# 📂 FileFlow — Linux File & Directory Automation Utility

**FileFlow** is a lightweight Bash-based file management utility that automates common Linux file and directory operations through an interactive command-line menu.

Designed as a practical Linux administration project, FileFlow demonstrates Bash scripting, filesystem management, user interaction, file permissions, and automation using standard Linux commands.

---

## ✨ Features

* 📄 Create executable files
* 🗂️ Create directories
* 🗑️ Delete files safely
* 📁 Delete empty directories
* 🔍 Locate files anywhere on the system
* 💬 Interactive menu-driven interface
* ⚡ Lightweight and easy to use
* 🐧 Compatible with most Linux distributions

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Kachinonye/FileFlow.git
```

Navigate to the project directory:

```bash
cd FileFlow
```

Make the script executable:

```bash
chmod +x fileflow.sh
```

---

## ▶️ Usage

Run the utility:

```bash
./fileflow.sh
```

You will be presented with an interactive menu.

Example:

```text
Choose an action:

1) Create File
2) Delete File
3) Create Directory
4) Delete Directory
5) Find File
6) Exit
```

Simply choose the desired option.
