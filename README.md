# Technology File Automation Script

This is a Bash script that automates basic file and directory operations via a simple menu interface.

## 🔧 Features

- Create a file (`technology.sh`)
- Delete the file
- Create a directory (`technology`)
- Delete the directory
- Find the full location of the file
- Exit the script

## 🖥️ Usage

1. Make the script executable:
   ```bash
   chmod +x technology.sh
